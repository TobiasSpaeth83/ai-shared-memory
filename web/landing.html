Collaboration Workflow

  1. Task Creation

    Tasks are added to context.json with owner assignment (claude, chatgpt, human)

  2. Agent Processing

    Assigned agent picks up tasks with status "todo" and begins implementation

  3. PR Creation

    Changes are submitted via pull requests with automated checks

  4. Review & Merge

    Code review by CODEOWNERS, CI checks pass, then merge to main

  5. State Update

    Task status updated to "completed" in shared memory
